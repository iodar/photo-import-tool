# photo-import-tool 

Werkzeug zum Importieren von Fotos. Features:
* Erzeugen einer Ordner Hierarchie mit Aufnahmedatum
* Abgleich von Fotos beim Import
* Lesen und Nutzen von EXIF-Daten
